About This Project

Project Overview

This project uses deep learning and transfer learning techniques to detect and classify Diabetic Retinopathy (DR) from retinal fundus images. Diabetic Retinopathy is a common complication of diabetes mellitus that can lead to blindness if not detected and treated early.

Technology Stack

Deep Learning

TensorFlow, Keras

Transfer Learning Models

Xception, ResNet50, Inception V3

Web Framework

Flask (Python)

Image Processing

OpenCV, Pillow

How It Works

  1. Upload a retinal fundus image through the web interface
  2. The image is preprocessed and normalized
  3. A pre-trained deep learning model analyzes the image
  4. The model classifies the image into one of 5 DR severity levels
  5. Results are displayed with confidence scores

Model Accuracy

The model achieves an average accuracy of 76.3% in classifying diabetic retinopathy severity levels.

Purpose

This project aims to provide a cost-effective and accessible solution for early detection of Diabetic Retinopathy, helping to reduce the risk of vision loss through prompt identification and treatment.
