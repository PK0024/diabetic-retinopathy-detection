Early Detection of Diabetic Retinopathy

Using Deep Learning and Transfer Learning Techniques

Diabetic Retinopathy (DR) is a common complication of diabetes mellitus that can lead to blindness if not detected early. Our AI-powered system uses advanced deep learning models to classify retinal fundus images into 5 severity levels, helping in early detection and treatment.

Get Started

Key Features

5-Level Classification

Classifies images into 5 stages from no DR to proliferative DR

Transfer Learning

Uses pre-trained models (Xception, ResNet50, Inception V3) for better accuracy

Fast & Accurate

Quick diagnosis with high accuracy to aid early detection

Easy to Use

Simple web interface for uploading and analyzing fundus images
